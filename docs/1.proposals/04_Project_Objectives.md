# Project Objectives

## Primary Objective

To develop a complete computational biology pipeline for multi-cancer type classification using TCGA RNA-Seq gene expression data while comparatively evaluating classical machine learning and deep learning approaches implemented entirely in Python.

## Specific Objectives

1. Acquire authentic TCGA-derived transcriptomic data through UCSC Xena.
2. Understand the biological and experimental foundations of RNA-Seq gene expression data.
3. Perform comprehensive data quality assessment and preprocessing.
4. Conduct exploratory analysis of high-dimensional transcriptomic datasets.
5. Apply statistical preprocessing and feature engineering techniques.
6. Develop multiple machine learning classifiers using scikit-learn.
7. Design and implement neural network models using PyTorch.
8. Compare predictive performance using standardized evaluation metrics.
9. Interpret computational findings within a biological framework.
10. Produce a reproducible GitHub repository and a comprehensive academic report documenting the complete workflow.

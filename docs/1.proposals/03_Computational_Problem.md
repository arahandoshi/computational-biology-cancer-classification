# Computational Problem

The objective of this study is to develop reproducible computational models capable of accurately classifying multiple cancer types using high-dimensional TCGA RNA-Seq gene expression data.

Each biological sample consists of thousands of quantitative gene expression measurements representing transcript abundance. These measurements collectively form a high-dimensional feature space in which complex linear and nonlinear relationships exist between gene expression patterns and cancer phenotypes.

To address this challenge, the project investigates two complementary predictive paradigms:

* Classical machine learning implemented using scikit-learn.
* Deep learning implemented using PyTorch.

Both approaches will be evaluated using standardized performance metrics, computational efficiency, robustness, reproducibility, and biological interpretability. The comparative analysis aims to identify strengths and limitations of each methodology within the context of transcriptomic cancer classification.

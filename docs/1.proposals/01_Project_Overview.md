# Project Overview

## Project Title

**Development and Comparative Evaluation of Machine Learning and Deep Learning Models for Multi-Cancer Type Classification from TCGA RNA-Seq Gene Expression Data**

## Overview

Cancer is a biologically heterogeneous disease characterized by distinct molecular signatures across different tissue types. Advances in high-throughput RNA sequencing have enabled comprehensive measurement of gene expression, generating large-scale transcriptomic datasets that provide valuable opportunities for computational analysis.

This project aims to develop a complete computational biology pipeline capable of classifying multiple cancer types using authentic TCGA-derived RNA-Seq gene expression data obtained through UCSC Xena. The study combines scientific computing, statistical analysis, classical machine learning, and deep learning within a reproducible Python ecosystem.

The project emphasizes rigorous computational methodology, reproducibility, scientific documentation, and biological interpretation. Beyond predictive modeling, the workflow is designed to strengthen practical expertise in transcriptomic data analysis while demonstrating advanced proficiency in NumPy, Pandas, Matplotlib, Seaborn, scikit-learn, and PyTorch.

The final outcome will be a fully documented computational biology repository accompanied by a comprehensive academic report describing the biological rationale, computational methodology, experimental evaluation, and interpretation of results.
